# SysP_3-1
## 16To2
### 설명
+ 무한으로 N 비트의 16진수를 입력받아 N*4 비트의 2진수로 전환시킴
+ N비트만큼 입력해야 함
+ "q"를 입력시 quit
### EX
4비트의 16진수를 16비트의 2진수로 전환
1. 0001 -> 0000000000000001 
2. 00A3 -> 0000000010100011
3. FFFF -> 1111111111111111
### 작동 원리
1. 16진수를 비트연산자인 ">>"을 통해 옆으로 한개씩 옮김
2. mask field인 "1"에 비트연산자인 "&"를 통해 계산
