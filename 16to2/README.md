# 16To2
## 설명
N 비트의 16진수를 N*4 비트의 2진수로 전환시킴
## EX
4비트의 16진수를 16비트의 2진수로 전환
1. 0001 -> 0000000000000001 
2. 00A3 -> 0000000010100011
3. FFFF -> 1111111111111111
## 작동 원리
16진수를 비트연산자인 ">>"을 통해 옆으로 한개씩 옮기고 mask field인 "1"를 만들어 비트 연산자인 "&"를 통해 계산
